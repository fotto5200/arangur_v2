# Arangur Plan Check Visual Concepts v1

Status: Three concepts for review; none selected
Depends on: `arangur_ui_client_report_how_am_i_doing_packet_v1.md` and `arangur_ui_selected_direction_v1.md`
Scope: Client-facing visual storytelling; not application navigation or implementation

## Shared sample

All three concepts present the same Northstar Family Office plan check for the twelve months ended June 30, 2026.

### Supplied or derived evidence

- Portfolio return: 7.79%.
- Policy benchmark: 6.99%.
- Ending portfolio value: approximately $44.9M.
- Client cash paid: $1.1M.
- AI / Chip Selloff: -$9.5M / -21.1%, leaving $35.4M.
- Rate Shock: -$4.8M / -10.7%, leaving $40.1M.
- Manager implementation effect: +1.05 percentage points.
- Advisor policy effect: -0.26 percentage points.

### Design-only objective and reconciliation assumptions

The following values exist only to make the visual concepts realistic. They are not assertions about supplied client data and must be replaced by validated production calculations:

- Client return objective: 6.5%.
- Client net asset-growth objective: 4.0% after distributions.
- Opening portfolio value: $42.7M.
- Capital appreciation: approximately $1.9M.
- Investment income: approximately $1.4M.
- Client distributions: -$1.1M.
- Fees/other reconciliation: approximately -$45K.
- Ending value: approximately $44.9M.
- Net asset growth: approximately 5.2%.
- Client downside review boundary: -15%, or approximately $38.2M from the current base.
- Illustrative probabilistic 80% range: $38.7M to $48.0M over the selected horizon.

The figures are internally coherent at displayed precision. The visual language must clearly distinguish verified outcome, client objective, deterministic scenario, and illustrative probability range.

## Shared presentation sequence

Every concept uses the same four-state Plan Check:

1. **Overall plan check** — Return and Capital Growth are ahead of objective; Resilience requires discussion.
2. **Return and capital growth** — Explain the difference between percentage return, capital appreciation, income, distributions, and ending assets.
3. **Resilience** — Compare current value, client boundary, deterministic scenarios, and probabilistic range on one scale.
4. **Discussion close** — Preserve the growth plan while reviewing AI concentration and the 10.5% unclassified exposure.

Cash Support remains available from the growth state as secondary evidence and can become a primary objective in a distribution-led briefing.

## Concept A — Objective Horizon

### Visual premise

Translate each active client objective onto a directly labeled horizontal measure: actual result, objective, and boundary. The opening reads as one plan with three aligned tests, not three dashboard cards.

### Above the fold

- Conclusion: “Growth is ahead of plan. One risk is outside the review boundary.”
- Return: 7.8% versus 6.5% objective.
- Net asset growth: 5.2% versus 4.0% objective.
- AI / Chip Selloff: -21.1% versus -15.0% review boundary.
- Dominant action: **See how capital grew**.

### State behavior

- The growth state replaces the objective measures with a concise capital reconciliation.
- Explain shows the advisor/manager contribution in one sentence.
- Verify shows the exact reconciliation or attribution source in one line before entering analytical evidence.
- The resilience state uses a single portfolio-value number line for both deterministic events and the probability range.
- The close leaves only the recommended discussion and review trigger.

### Motion

When advancing, actual markers move from the objective measures into the capital reconciliation. Scenario markers move along a fixed value axis; the axis never rescales between cases. Motion stops after each transition and honors reduced-motion preferences.

### Strengths

- Fastest objective-versus-result comprehension.
- Makes exception status legible without a red/amber/green scorecard.
- Scales well across different client objective selections.
- Stable scenario scale supports trustworthy comparison.

### Failure modes

- Repeated measures can feel analytical rather than emotionally engaging.
- The opening can become a KPI rail set if each objective gains secondary labels.
- Normalizing unlike objectives may imply they are mathematically comparable.
- Too many client objectives would destroy the composition.

### Simplification requirement

Keep only actual, objective, and one directly stated consequence per measure. Remove legends, mini-trends, badges, and secondary benchmarks from the opening.

## Concept B — The Capital Landscape

### Visual premise

Use one continuous capital path as the visual world of the briefing. The client sees the plan path, actual outcome, sources of capital change, and forward risk as transformations of the same portfolio—not as separate charts.

### Above the fold

- Conclusion: “Northstar finished above its growth path; the forward range is wider than the agreed boundary.”
- One continuous line from $42.7M opening value to $44.9M ending value.
- The client’s plan path ends below the actual value.
- The AI risk boundary is visible ahead but not yet expanded into scenarios.
- Dominant action: **Follow the capital**.

### State behavior

- The growth state annotates the path with appreciation, income, distribution, and reconciliation effects.
- Explain isolates the 7.79% flow-adjusted return from the 5.2% change in total assets.
- Verify exposes the exact starting value and capital-flow reconciliation.
- The resilience state extends from the same $44.9M present point into the probabilistic range and deterministic scenario endpoints.
- The close returns to the plan path and identifies the proposed review rather than ending on a loss scenario.

### Motion

The portfolio marker advances from opening to ending value. Capital effects alter the path sequentially. Forward ranges grow from the fixed current point; deterministic scenarios branch only when Resilience is entered. Exported versions use the same sequence as adjacent static frames.

### Strengths

- Most intuitive explanation of capital appreciation versus asset growth.
- Treats distributions as part of the wealth story rather than unexplained loss.
- Creates meaningful continuity between historical result and forward risk.
- Gives animation a clear explanatory role.

### Failure modes

- Can drift into the previously rejected Wealth Journey narrative if objective status stops leading.
- Forecast lines can imply false precision.
- Complex capital flows may overwhelm the single path.
- Requires fully reconciled start/end values and reliable valuation dates.

### Simplification requirement

The objective conclusion remains visible in every state. Show no more than four capital effects, group immaterial flows, and distinguish observed history from future uncertainty by form and label—not decoration.

## Concept C — Editorial Focus

### Visual premise

Treat the briefing like a tightly edited presentation: one declarative sentence and one hero visual relationship at a time. The opening prioritizes comprehension and presence over simultaneous comparison.

### Above the fold

- Statement: “Ahead on growth. Outside one risk boundary.”
- One directly labeled relationship pairs 7.8% return with the 6.5% objective.
- Capital Growth and Resilience appear as the next two chapters, not peer panels.
- Dominant action: **See capital growth**.

### State behavior

- Return receives one state and one objective comparison.
- Capital Growth receives one state showing $42.7M opening, approximately $1.9M appreciation, and $44.9M ending assets after flows.
- Resilience receives one shared-scale scenario statement and visual.
- Explain temporarily replaces the hero caption with one concise driver statement.
- Verify opens evidence from the exact visible claim.
- The close is a single advisor recommendation with an explicit monitoring trigger.

### Motion

The objective marker transitions to the actual result rather than counting from zero. Capital-growth components rearrange into ending value. Scenario outcomes move from the same current-value position. No content loops or autoplays after arrival.

### Strengths

- Calmest and most presentation-ready concept.
- Strongest chance of producing an immediate “I get it” response.
- Excellent for five-minute clients and advisor-led conversation.
- Prevents the opening from becoming a dashboard.

### Failure modes

- Requires more presentation steps to cover the same evidence.
- Sophisticated clients may perceive it as withholding information.
- Large declarative language can overstate certainty if caveats are not disciplined.
- The shared plan status may become less obvious as each objective gets its own chapter.

### Simplification requirement

Each state contains one sentence, one visual relationship, and one primary action. Secondary facts are removed rather than placed in supporting cards.

## Review boundary

Select one visual premise before polished mockups. Do not combine Objective Horizon’s simultaneous measures, Capital Landscape’s continuous path, and Editorial Focus’s chapter model into a single compromise screen.

# Arangur UI Selected Direction v1

Status: Selected for visual exploration
Selected: July 13, 2026

## Scope of selection

**The Plan Check** is selected as the storytelling direction for the first client-facing “How am I doing?” report.

This selection applies to the report narrative only. It does not select one of the unresolved application-level interaction architectures in `arangur_ui_interaction_directions_v1.md`.

## Selected premise

The report is organized around the objectives that define success for the client. Arangur supports Return, Capital Growth, Cash Support, and Resilience, while an individual opening promotes no more than two or three relevant objectives.

## Pilot profile

- Client: Northstar Family Office.
- Setting: advisor-led five-minute presentation.
- Reporting emphasis: growth-oriented.
- Opening objectives: Return, Capital Growth, and Resilience.
- Cash Support: available as a secondary client discussion, not an opening peer.
- Depth contract: Understand → Explain → Verify → return to the same conclusion.

## Fixed design boundaries

- One conclusion and one dominant action per presentation state.
- No more than three primary figures per state.
- Client objective precedes benchmark comparison.
- Return, capital appreciation, asset growth, and cash support remain distinct.
- Scenario comparisons use a consistent portfolio base, value scale, and horizon.
- Deterministic and probabilistic outcomes remain visibly different.
- Tables, methodology, and position-level calculations remain at verification depth.
- Animation explains change, comparison, or causality and has a meaningful static equivalent.
- Client presentation contains no report-generation, template, data-loading, export, or administration controls.

## Next stage

**Objective Horizon** is the preferred concept for continued refinement. This is not yet a frozen visual system. The refinement must test stable favorable-right/unfavorable-left direction, portfolio-to-manager drill-down, and applicability beyond this report before polished mockups or prototype work.

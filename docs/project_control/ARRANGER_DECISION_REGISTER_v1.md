# Arranger Decision Register v1

Status: consolidated from committed repo evidence and external design inputs on 2026-07-15

Status meanings:

- **Accepted**: durable repo direction or implemented product rule.
- **Provisional**: active direction with remaining design/method approval.
- **Superseded**: retained for reference but no longer the primary product surface.
- **Needs Frank review**: unresolved product choice or external design input not yet integrated into the repo.

## Superseded/reference decisions to preserve

- Policy-Level Attribution Summary v1: retained as a calculation/reference artifact; not the primary product-review surface.
- Dense reporting console and raw artifact browser as the main advisor UI: implementation history only.
- One-report guided wizard as the target model: superseded by briefing/workflow composition.
- A single mixed-category concentration table: superseded by one consistent category system per report.
- Cash-Flow Support Readiness as a client report: replaced by separate Cash Flow Delivered and Cash-Flow Support Outlook reports when prerequisites exist.

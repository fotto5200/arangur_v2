# Imputed Baseline Policy Attribution Variant

When current manager weights are accepted as the baseline, allocation drift effect is suppressed and manager implementation remains measurable.

## Key Metrics

- **Allocation drift effect:** 0.00 pp (suppressed)
- **Baseline:** Current manager weights
- **Manager implementation effect:** +1.05 pp

## Imputed Baseline Variant

| Item | Status | Meaning |
| --- | --- | --- |
| Baseline | Current manager weights | Actual weights are treated as target weights for setup review. |
| Allocation drift effect | Suppressed | Target-versus-actual policy drift is not shown in this variant. |
| Manager implementation effect | Available | Actual manager returns are compared with mandate benchmark returns. |
| Use | Advisor setup | This variant is not the standard client report and does not prove the current allocation is ideal. |

## Caveat

- This variant is a setup/readiness view, not proof that the current allocation is ideal.

## Advisor Note

Use the explicit policy report when advisor/family target weights are available.
